Usage: waCron.cmd [COMMAND] [PARAMETERS]

- will execute [COMMAND] [PARAMETERS]
- if ERRORLEVEL is not 0, STDOUT or STDERR are not empty
  then log error to file (waCron_09_03_2004.log)
  Optionally, let script send email/sms/...
- to be used in Scheduled Tasks or AT schedules
- waCron.cmd /ver will give version and license info
